Barnowl Stuffy Demo
===================

Web visualisation of the barnowl stuffy with accelerometer beacon.


Install and Run
---------------

Clone the repository, then:

    npm install
    node server

and browse to [localhost:3000](http://localhost:3000).  You'll need a Minew i7 accelerometer beacon for the barnowl stuffy to appear and change orientation.
